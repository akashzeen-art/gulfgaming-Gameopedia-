## API Configuration

**Base URL:** `https://prod.in.asgard.apis.simpleviralgames.com/game/category/`

**Headers:**
```
client-id: 50202bba-0a97-40cf-b980-7b62b1cfad0e
```

**Query Parameters:**
- `p=1` - Page number
- `platform=B2BW`
- `language=en`
- `multiplayer=false`
- `access_token=null`
